# Store-Data-Analysis-using-Excel-
Data Analytics Project

Store Annual Report 2022 – Excel Data Analysis Project
Project Description
This project is an **interactive Excel dashboard** built to analyze and visualize the **Store Annual Sales Data for the year 2022**.  
The dashboard converts raw transactional data into meaningful insights using **Pivot Tables, Charts, and Slicers**.

The goal of this project is to help business users understand **sales trends, customer behavior, order performance, and channel contribution** in a single view.


 Business Questions Answered
- Compare **Sales and Orders** using a single chart
- Identify the **month with the highest sales and orders**
- Determine **who purchased more – Men or Women**
- Analyze **order status distribution**
- Identify **top-performing states**
- Understand the **relationship between age group and gender**
- Find which **sales channel contributes the most**
- Identify the **highest selling product category**


 Dataset Information
The dataset contains **store sales data for 2022**, including:

- Month
- Order ID
- Sales Amount
- Number of Orders
- Gender (Men / Women)
- Age Group (Adult, Senior, Teenager)
- Order Status (Delivered, Cancelled, Returned, Refunded)
- Sales Channel (Amazon, Flipkart, Myntra, Meesho, Ajio, Nalli)
- State
- Product Category (Kurta, Saree, Blouse, Bottom, Ethnic Dress)


 Key Insights Summary
- Women contribute **64% of total sales**
- **92% orders are successfully delivered**
- **Amazon** is the highest contributing sales channel
- **Maharashtra** is the top sales-generating state
- **Adults** are the most active customer segment
- Sales peak during **March**
